## [0.1.0] - 2025-02-06

### Added
- Initial specification release
- Five emotional primitives (frustration, trust, urgency, stress, caution)
- Linear decay semantics (0.05 per 300 seconds)
- Standard event registry (13 events)
- MCP tool definitions (4 required tools)
- Conformance test vectors
- Action selection with precedence hierarchy
- JSON schemas for state, events, and MCP tools
- Governance and RFC process documentation

### Scope
- v0.1 is a **draft specification**
- Breaking changes may occur before v1.0
- Feedback and iteration expected
